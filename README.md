# Activity 3 Extra: BDD with Cucumber and Playwright (Swag Labs)

โปรเจกต์นี้เป็นงานสำหรับฝึกทำ **BDD (Given–When–Then)** ด้วย **Cucumber** และทำ **E2E automation** ด้วย **Playwright** บนเว็บตัวอย่าง **SauceDemo (Swag Labs)** โดยทดสอบฟีเจอร์ **Add/Remove Cart** จำนวน 4 scenarios (2 happy, 1 negative, 1 edge)
